# ProyectoFinal
Contenido
La página de Hey Tenis consta de los siguientes elementos:

Página de inicio: Aquí se presenta una introducción a nuestras clases de tenis, destacando los beneficios, el enfoque de enseñanza y los horarios disponibles. Además, se incluyen testimonios de estudiantes satisfechos y se muestra una galería de imágenes de nuestras instalaciones.

Información de clases: En esta sección, se proporciona información detallada sobre los diferentes tipos de clases que ofrecemos, como clases grupales para principiantes, entrenamiento avanzado, clases para niños, etc. Cada tipo de clase se describe brevemente, incluyendo los objetivos, los niveles de habilidad recomendados y los horarios disponibles.

Tecnologías utilizadas
La página de Clases de Tenis se desarrolló utilizando las siguientes tecnologías:

HTML: Para la estructura y el contenido de la página web.
CSS: Para el diseño y la presentación visual de la página.
JavaScript: Para la interactividad y la funcionalidad dinámica, como el calendario interactivo y la validación de formularios.
